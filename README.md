# Democratic Machine Learning System

A comprehensive governance system for the United States with research-backed implementation.

## Repository Structure

```
democratic_machine_learning/
├── src/                    # Core Python modules
│   ├── core/              # Decision engine, weighting, feedback
│   ├── data/              # Data loading, preprocessing, features
│   ├── models/            # Data models (Voter, Policy, Region, Decision)
│   ├── ui/                # TUI interface
│   ├── utils/             # Metrics, validation, logging
│   ├── security/          # Trust system, anti-pattern detection
│   └── history/           # Anti-pattern database
├── research/              # Research files with citations
├── output/                # Generated analysis outputs
├── AGENTS.md             # Development guidelines
├── ARCHITECTURE.md       # System architecture
├── EXECUTIVE_SUMMARY.md  # Executive overview
├── TUTORIAL.md           # Usage guide
└── README.md             # This file
```

## Quick Start

```bash
# Run governance reasoning engine
python3 governance_reasoning_engine.py

# Run real execution system
python3 real_execution_system.py

# Run immigration policy evaluator
python3 run_immigration_eval.py

# Run election analysis
python3 run_election_analysis.py
```

## Key Components

### Governance Reasoning Engine
- Uses research files to design governance systems
- Applies 7 core principles (Inclusivity, Fairness, Transparency, etc.)
- Generates comprehensive reasoning with historical context

### Real Execution System
- Collects real data from Pew Research, Gallup, Census, BLS
- Performs 12×12 cross-reference analysis
- Generates policy recommendations with confidence scores

### Anti-Pattern Database
- Historical governance failures documented
- Detection metrics for each anti-pattern
- Mitigation strategies for prevention
