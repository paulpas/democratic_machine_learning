## Appendix: Voting Method Details

### Weight Calculation Formula

```
Weight(citizen) = Base_Weight × (1 + Expertise_Boost × 0.40) × (1 + Proximity_Boost × 0.35) × (1 + Participation_Boost × 0.25) × (1 + Historical_Redress × 0.15)
```

Where:
- **Base_Weight** = 1.0
- **Expertise_Boost** = 0-1 based on demonstrated expertise
- **Proximity_Boost** = 0-1 based on geographic impact
- **Participation_Boost** = 0-1 based on past participation
- **Historical_Redress** = 0.15 for historically marginalized groups

### Approval Voting Procedure

1. Each citizen reviews each policy
2. Citizen approves or disapproves of each policy
3. Weights are applied to approvals
4. Policies with weighted approval > 50% pass
5. Fairness constraints verified for all affected groups

### Fairness Constraint Verification

```
For each demographic group G:
  satisfaction(G) = weighted_approvals(G) / total_weight(G)
  if satisfaction(G) < 0.30: policy rejected
  if max(satisfaction) - min(satisfaction) > 0.40: policy requires revision
```
